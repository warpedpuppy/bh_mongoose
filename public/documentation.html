MyFlix

The objective of this api is to focus on the server side componenets of a movies web application. The web application will provide information about movies, directors and different genres to users. Users will have the option to sign up with their personal information and create a list of their favorite movies.

Request Url Http Method Request Body Response Body
Get list of Movies /movies GET None Json objects show data on list of movies
Movie details /movies/details/:title GET None Json objects showing data on movie title, img, description,genre, directors, actors
Movie Directors /movies/directors/:name Get None Json objects returns information on directors, birth, death
Delete movies from list /movies/remove/:title Delete None message indicating the status on if the movie was deleted or not
Add a New User /users Post Json about new username Json with information added and id confirming completion or failure
Log in /users Get None Json with data about users account
Update account Post /users Json with data that is being added Json confirming that updates have been applied or denied
Delete a user /users/delete/:username Delete None Message indicating the status of whether or not the request was completed
